## Customization

### Personal Information
1. **Name & Title**: Update in `index.html`:
   - Line 11: Change `<title>` tag
   - Line 30: Update "Your Name" in hero section
   - Line 31: Update subtitle/role

2. **Contact Information**: Update in `index.html`:
   - Lines 200-212: Update email, phone, and location

3. **About Section**: Customize in `index.html`:
   - Lines 60-75: Update about text and statistics

4. **Skills**: Modify in `index.html`:
   - Lines 85-120: Update skill cards with your expertise

5. **Projects**: Update in `index.html`:
   - Lines 130-165: Add your actual projects

### Colors & Styling
Edit `styles.css` to customize:
- Primary colors: Change `--primary-color` and `--secondary-color` in `:root`
- Fonts: Update Google Fonts import in `index.html` line 8
